Selective Theory Sorter — Self-Grading Version

This version keeps the same -3 to +3 response scale as your attached document, but it automatically totals the columns, maps them to theories, and identifies the top scoring approaches.

How to use it

For each statement, choose a number from -3 to +3, where -3 means “not at all like me,” 0 means neutral, and +3 means “a lot like me.”

-3 strongly unlike me 0 neutral +3 strongly like me
